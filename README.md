#  PawHome+ : Smart Pet Rehoming, Rescue & Donation Platform

An interactive full-stack web application that connects pet donors, adopters, and shelters to enable safe, ethical, and efficient pet rehoming while preventing pet abandonment.

---

## Overview

PawHome+ provides a reliable platform where users can rehome pets, adopt responsibly, and support rescue efforts. The system ensures transparency, trust, and long-term pet well-being.

---

## Features

* Pet rehoming and adoption system
* Browse pets with filters (type, age, location)
* Wishlist functionality
* Detailed pet profiles
* Responsive UI (mobile + desktop)
* Real-time chat between adopter and donor
* AI chatbot for pet care guidance
* Notifications for updates and alerts
* Emergency rehoming mode
* Smart pet matching system
* Guided pet transition support
* Post-adoption communication
* Donation system with predefined and custom options
* Donation tracking dashboard
* User verification and safety system
* Ratings, reviews, and report system
* Pet health and history tracking

---

##  Tech Stack

* **Frontend:** React / Next.js
* **Backend:** Node.js + Express
* **Database:** MongoDB
* **Authentication:** JWT / Firebase
* **Real-time:** Socket.IO / Firebase
* **Image Upload:** Cloudinary

##  How It Works

* Users list pets with complete details
* Pets appear in listings with filters
* Adopters browse and select pets
* Real-time chat enables communication
* System ensures safe and transparent adoption process
